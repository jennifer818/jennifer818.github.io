Introduction

This project explores diffusion models and using them to generate images. In part A, I will employ a pretrained denoising neural net, DeepFloyd, in order to iteratievly denoise and generate images. In part B, I will implement and train my own denoiser in order to generate images with numbers on them. Note: each notebook in this project has global seed=180

Part A: The Power of Diffusion Models!

For this part of the project, I will use a pre-trained diffusion model to denoise images, generate images from purse noise, editing images, inpaint images, create visual anagrams, and make hybrid images.

A: 1.6. Classifier-Free Guidance (CFG)

To improve the generated images, we can use classifier-free guidance. The only difference from iterative denoising is that the noise estimate incorporates both a conditional noise estimate (noise estimate from diffusion model using a prompt embedding i.e. "a high quality photo") and an unconditional noise estimate (noise estimate from diffusion model using the embedding for an empty string, ''). The equation for the noise estimate is uncond_noise + gamma * (cond_noise - uncond_noise). I used a gamma of 5.

These generated images look much more realistic.

A: 1.8 Visual Anagrams

We can generate images that look like one thing one way and another thing flipped around by tweaking the noise estimate in iterative denoising. At each time step, we can get one pair of conditional/unconditional noise estimates by passing in the current image and one prompt embedding/unconditional embedding into DeepFloyd and another pair of noise estimates by passing in the flipped current image and the other prompt embedding/unconditional embedding. We then average these noise estimates and implement cfg as usual.

Note: I upsampled these images using on the two prompt embeddings so they are clearer.
